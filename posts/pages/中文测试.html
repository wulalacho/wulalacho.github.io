数据结构笔记-链表

链表的定义： 如图所示，单链表的节点分为数据域和指针域，可以将它们视为一个整体，称之为节点，稍后我们会用结构体来表示节点。 数据域，顾名思义，就是存放数据的地方。 指针域，是用来存放指针变量的地方，指针指向下一个节点的地址。

顺序表的特点
  • 随机访问，可以在$O(1)$时间内找到对应元素。
  • 存储密度高，只用于存储数据。
  • 拓展容量不方便
  • 插入删除操作不方便
  • 表中的元素逻辑地址与物理地址顺序相同。

顺序表定义

使用$C$语言的结构体定义顺序表，M-Mode 使用 typedef 定义一个ElemType表示数据基本类型， 并定义最大长度 MAXSIZE ： 
struct CPUState {
      // Processor internal state
      uint32_t regs[32], csrs[CSR_COUNT];
  
      // Memory state
      uint8_t *mem;
      uint32_t mem_offset, mem_size;
  };
